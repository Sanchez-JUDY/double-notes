# double-notes
一个简洁的计划与手记管理应用。
本项目为纯静态项目，无需安装依赖，
双击打开 index.html 即可运行。

# 📖 双记

> 一个简洁、优雅的纯原生 JavaScript 单页应用，用于管理日常计划和随手笔记。

#🌟 项目简介

大学生一线前端练手项目，"双记" 旨在不依赖任何重型框架的情况下，实现流畅的本地数据管理体验。项目采用 LocalStorage 实现数据持久化，支持计划任务的增删改查以及富文本手记的记录。

# ✨ 核心功能

# 📅 计划模块
- 任务管理：支持添加、删除计划任务。
- 状态筛选：支持“全部”、“进行中”、“已完成”三种状态切换。
- 视觉反馈：完成任务后自动添加删除线和置灰效果。

# 📝 手记模块
- 即时保存：基于 LocalStorage，数据自动保存在本地浏览器，刷新不丢失。
- 分类导航：自动根据时间生成年月分类标签，方便归档查找。
- 关键词搜索：支持对手记标题和内容进行实时模糊搜索。
- 响应式设计：完美适配移动端与 PC 端。

# 🛠️ 技术栈

- HTML：语义化标签
- CSS3：Flexbox 布局，响应式媒体查询，CSS 变量管理主题色。
- JavaScript (ES6+)：
  - 原生 DOM 操作与事件委托。
  - LocalStorage API 封装与数据持久化。
  - 模块化思维（Storage 模块独立封装）。
- Tailwind CSS：用于快速构建现代化 UI。
